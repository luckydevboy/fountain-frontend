Fountain is an e-commerce website with a focus on stationery.

### Technologies I've used

- Web Framework: [Next.js](https://nextjs.org/)
- CSS Framework: [Tailwind CSS](https://tailwindcss.com/)
- HTTP Client: [Axios](https://axios-http.com/docs/intro)
- Asynchronous State Management: [React Query](https://tanstack.com/query/latest)
- Auth Framework: [NextAuth](https://next-auth.js.org/)
- Motion Library: [Framer Motion](https://www.framer.com/motion/)

For the backend, Fountain uses [Strapi CMS](https://strapi.io/) hold in [this repo](https://github.com/luckydevboy/fountain-cms).
